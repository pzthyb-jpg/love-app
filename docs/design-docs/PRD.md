## 1. 产品概述

### 1.1 产品名称
**小皮爱情助手**（Love App）

### 1.2 产品定位
一款专为情侣设计的 **私密、轻量化 H5 恋爱助手应用**，以「拍照打卡 + 智能彩虹屁」为核心，辅以午餐转盘、愿望池、情话留言等功能，帮助情侣记录日常、增进感情、制造小惊喜。

### 1.3 核心价值
| 维度 | 价值描述 |
|------|----------|
| **情感连接** | 每日拍照打卡 + AI 彩虹屁，让对方感受到被关注和被夸赞 |
| **决策辅助** | 午餐转盘解决「今天吃什么」的世纪难题 |
| **愿望管理** | 愿望池记录彼此心愿，标记实现进度，制造仪式感 |
| **隐私安全** | 纯本地存储，数据不上传服务器，保护情侣私密空间 |
| **离线可用** | PWA 支持，无网络也能使用全部功能 |

### 1.4 技术栈
- **前端框架**: Vue 3.5 + Vue Router 4
- **构建工具**: Vite 8
- **UI 组件库**: Vant 4.10
- **PWA 支持**: vite-plugin-pwa (Workbox)
- **数据存储**: localStorage + IndexedDB
- **安全**: Web Crypto API (SHA-256 哈希)

## 4. 核心用户流程

### 4.1 拍照打卡流程
```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  打开摄像头  │ ──▶ │  实时预览   │ ──▶ │  拍照确认   │ ──▶ │  生成彩虹屁  │
│  (getUserMedia)  │  (video)    │     │  (canvas)   │      │  (模板引擎)  │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
                                                                   │
                                                                   ▼
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  成就检查   │ ◀── │  更新连续天数│ ◀── │  保存记录   │ ◀── │  展示结果   │
│  (里程碑)   │     │  (streak)   │     │ (IndexedDB) │     │  (Toast)    │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
```

**详细步骤**:
1. 用户点击「咔嚓拍照」按钮
2. 调用 `navigator.mediaDevices.getUserMedia()` 打开前置摄像头
3. 视频流渲染到 `<video>` 元素实时预览
4. 用户点击「咔嚓！拍照」按钮
5. 使用 Canvas 截取视频帧，镜像翻转后导出为 JPEG (质量 0.7)
6. 智能彩虹屁引擎根据时间段、节日、随机因子生成夸赞文案
7. 打卡记录保存到 IndexedDB（照片）+ localStorage（元数据）
8. 计算连续打卡天数，检查是否达成新里程碑
9. 达成里程碑时触发庆祝弹窗 + 彩纸动画 + 触感反馈

**彩虹屁生成逻辑**:
- **基础模板**: 颜值夸赞（15条）
- **时间段**: 早晨(4条) / 午间(4条) / 傍晚(4条)
- **随机追加**: 妆容(10条) 或 元气状态(9条)
- **特殊日**: 节日/纪念日追加特殊文案(5条)
- **昵称替换**: `{name}` 替换为用户昵称

### 4.2 午餐转盘流程
```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  点击转盘   │ ──▶ │  旋转动画   │ ──▶ │  结果展示   │
│  (Canvas)   │     │  (4s缓动)   │     │  (弹窗)     │
└─────────────┘     └─────────────┘     └─────────────┘
                                               │
                          ┌────────────────────┼────────────────────┐
                          ▼                    ▼                    ▼
                    ┌───────────┐        ┌───────────┐        ┌───────────┐
                    │  确认去吃  │        │ 再转一次   │        │  导航/收藏 │
                    │  (记录)   │        │ (排除当前) │        │  (高德)   │
                    └───────────┘        └───────────┘        └───────────┘
```

**转盘算法**:
- Canvas 绘制扇形区域，每个餐厅占 `(2π/n)` 弧度
- 旋转角度 = 随机 5-8 圈 + 随机偏移
- 缓动函数: `1 - (1-t)^4` (cubic-out)
- 结果计算: 指针角度(顶部) - 当前旋转角度 → 扇形索引

### 4.3 愿望池流程

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  输入内容   │ ──▶ │ 选择类型    │ ──▶ │  提交保存   │
│  (textarea) │     │ (许愿/吐槽) │     │ (localStorage)│
└─────────────┘     └─────────────┘     └─────────────┘
                                               │
                                               ▼
                                        ┌─────────────┐
                                        │  气泡展示   │
                                        │ (长按操作)  │
                                        └─────────────┘
                                               │
                          ┌────────────────────┼────────────────────┐
                          ▼                    ▼                    ▼
                    ┌───────────┐        ┌───────────┐        ┌───────────┐
                    │ 标记已实现 │        │   删除    │        │   无操作   │
                    │ (prompt)  │        │ (confirm) │        │           │
                    └───────────┘        └───────────┘        └───────────┘
```

### 4.4 留言管理流程
```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  连续点击   │ ──▶ │  密码验证   │ ──▶ │  管理界面   │
│  标题5次    │     │ (SHA-256)   │     │ (CRUD)      │
└─────────────┘     └─────────────┘     └─────────────┘
```

**安全机制**:
- 默认密码: `1314`
- 密码使用 SHA-256 哈希存储
- 支持 4 位数字密码修改
- 入口隐藏（首页标题连续点击 5 次）
