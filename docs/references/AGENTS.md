# 💕 小皮爱情助手 — 给女朋友的 H5 小程序

## 项目概述

这是一个专为女朋友设计的 H5 移动端网页应用，风格参考 Za Bank — 简洁、干净、可爱。
三个核心功能：拍照彩虹屁、午餐大转盘、愿望池。

## 技术栈

- **前端**: 纯 HTML5 + CSS3 + JavaScript（无框架依赖，零构建）
- **存储**: localStorage（愿望池、打卡记录持久化）
- **地图**: 预设餐厅列表（非真实 API），支持手动添加/删除
- **AI 引擎**: 内置彩虹屁生成引擎（30+ 基础模板，随机组合变体）
- **部署**: 静态网页，可部署至 Vercel / GitHub Pages / 任意静态托管

## 功能说明

### 1. 📸 拍照打卡 + 彩虹屁
- 中午定时提醒（浏览器 Notification API）
- 调用手机摄像头自拍
- 识别拍照时间与日期特征，生成独一无二的彩虹屁
- 每日打卡记录保存在 localStorage

### 2. 🍽️ 午餐大转盘
- 预置 10 家精选餐厅（含距离、评分、标签）
- Canvas 绘制转盘，随机选择吃什么
- 支持手动添加/删除/重置餐厅列表
- 转盘历史记录保存在 localStorage

### 3. ✨ 愿望池
- 提交愿望或吐槽内容
- 展示为彩色气泡（浮动动画）
- 持久化到 localStorage

## 架构约束

- **单向依赖**: css/ 无依赖，js/ 模块间不互相 import（通过全局命名空间）
- **浏览器 API**: 仅使用标准 Web API（Camera、Notification、localStorage）
- **无后端**: 纯前端应用，零服务端依赖
- **移动优先**: 以 375px-414px 宽度为设计基准
