### TC-H001：添加随机类型留言

| 项目 | 内容 |
|------|------|
| **编号** | TC-H001 |
| **名称** | 添加随机类型留言 |
| **严重级** | P0 |
| **前置条件** | 1. 用户已登录（testuser）<br>2. 已输入正确管理密码<br>3. 当前在 `/messages-admin` 页面 |

**操作步骤：**

1. 点击「✨ 添加留言」按钮
2. 页面跳转到 `/messages-admin/new`
3. 在留言内容输入框输入 `今天也要开心哦！`
4. 确认类型选择为「🎲 随机」（默认）
5. 确认触发条件选择为「🚫 无」（默认）
6. 点击「保存」按钮
7. 观察 Toast 提示
8. 页面自动跳转回 `/messages-admin`

**预期结果：**

- 页面标题显示「添加留言」
- 输入框实时显示输入内容
- 点击保存后弹出 Toast「留言已添加」（绿色成功样式）
- 页面自动跳转回留言管理列表
- 列表中出现新添加的留言，类型为「🎲 随机」
- `dataStore.state.messages` 数组长度 +1
- 新留言对象包含字段：`text: '今天也要开心哦！'`, `type: 'random'`, `specialCondition: null`, `author: '男朋友'`
- Supabase `messages` 表中新增一条记录，`content` 字段为 `今天也要开心哦！`

**验证方式：**

- [ ] UI 视觉检查
- [ ] Vue DevTools 检查 dataStore.state.messages
- [ ] 网络请求检查 POST /messages 返回 200
- [ ] 刷新页面后留言仍存在
