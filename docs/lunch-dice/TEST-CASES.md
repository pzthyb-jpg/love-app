## 一、测试范围

本次迭代覆盖以下新功能：
1. 3D 台球桌面 + 骰子渲染
2. 陀螺仪摇晃检测
3. 底部抽屉 + 餐厅拖拽排序
4. 骰子掷出动画 + 结果映射
5. 兜底方案（无陀螺仪设备）

## 五、风险

| 风险 | 影响 | 缓解 |
|------|------|------|
| iOS 权限请求被拒绝 | 无法使用陀螺仪 | 降级为点击触发 |
| 低端安卓 CSS 3D 卡顿 | 动画不流畅 | 提供 prefers-reduced-motion 降级 |
| 拖拽排序在 iOS Safari 有 300ms 延迟 | 交互不跟手 | 使用 touch-action: none |
