## 6. 数据流更新示意

### RED-1 快速打卡数据流

```
用户点击「立即打卡」
    │
    ▼
Home.vue: quickCheckin()
    │ 调用
    ▼
dataStore.addQuickCheckin(todayStr)
    │ 写入
    ▼
state.checkinHistory.unshift({ date: today, type: 'quick' })
    │ + 持久化
    ▼
safeSetJSON(STORAGE_KEYS.CHECKIN_HISTORY, state.checkinHistory)
    │
    ▼
UI 自动响应式更新:
  - todayCheckedIn = true
  - 按钮 disabled
  - 状态圆圈变实心
```

### RED-2 动画密度数据流

```
设置页选择档位
    │
    ▼
Settings.vue: onDensityChange(val)
    │ 写入
    ▼
localStorage.setItem('animation_density', val)
    │ + DOM 同步
    ▼
document.documentElement.dataset.animation = val
    │
    ▼
Home.vue CSS 根据 [data-animation="..."] 选择器:
  - off     → .floating-hearts { display: none }
  - compact → 3 颗，透明度 0.4，周期 12s
  - full    → 6 颗（默认）
  - density → 12 颗，速度 +30%
```
